Adding a hero slide

The homepage hero slider utilises the bootstrap carousel. You can create a new slide by including the HTML below on line 55 of index.html

<div class="item" style="background-image: url(images/hero-3.jpg)">
	<div class="container">
		<div class="row blurb scrollme animateme" data-when="exit" data-from="0" data-to="1" data-opacity="0" data-translatey="100">
			<div class="col-md-10 col-md-offset-1">
				<h1>The perfect template to compliment</h1>
				<h2>Your brewery</h2>
				<a href="#" class="btn btn-default">
					<span>Buy template</span>
				</a>
			</div>
		</div>
	</div>
</div>

Adding an owl carousel

The owl carousel is used for items such as product sliders. You can add multiple carousels to the tamplate using the HTML below. You can also add a unique ID which allows you to modify the sliders ettings vis javascript.

<div class="owl-carousel" id="MyCarousel">
	<!-- Products -->
	<div class="product item">
		<a href="our-beer.html">
			<span>Buy it</span>
			<img src="images/can-1.png" alt="Beer can mockup">
		</a>
		<h3>Pale ale</h3>
		<h4>$4.75</h4>
	</div>
</div>

Once you have included your HTML, you will then need to include the following JavaScript to enable the carousel.

$("#myCarousel").owlCarousel({
	loop: true,
	autoplay: true,
	autoplayTimeout: 4000,
	autoplaySpeed: 500,
	autoplayHoverPause: true,
	responsive : {
		480 : { items : 1  },
		768 : { items : 3  },
		1024 : { items : 4 }
	},
});

Parallax

Craft Beer Nation makes use of Scrollme which creates awesome parallax effects for on-page elements. Please take a look at the plugin homepage for more examples of how to use this as well as the examples within the HTML template.

You can add parallax elements by including two classes and a set of data attributes like so.

<div class="scrollme animateme" data-when="exit" data-from="0" data-to="1" data-opacity="0">
